# About This Workshop

## Overview

This workshop is designed to build practical and conceptual mastery of Cisco
Secure Firewall (FTD) — from foundational packet flow analysis to advanced deep
dive configurations. Participants will learn;

**Key Topics:**

- Understand the FTD architecture and traffic flow sequence.
- Master NAT types, translation order, and verification techniques.
- Configure and troubleshoot Access Control Policies (ACP).
- Design and deploy Site-to-Site and Remote Access VPNs using IKEv2, SSL, and VTIs.
- Use FTD CLI and FMC for end-to-end visibility and diagnostics.

## Purpose

This workshop bridges theory and practice — enabling network and security
professionals to apply Cisco Secure Firewall skills confidently in real-world
scenarios. Through guided labs, students will gain deep operational insight into
packet handling, policy decisions, NAT behavior, and VPN connectivity.

**Target Audience:**

- Network and Security Engineers
- System Integrators and Support Professionals
- Students preparing for Cisco certifications (CCNP Security / CCIE Security)
- IT teams adopting Cisco Secure Firewall solutions in enterprise environments

## Learning Objectives

By the end of this workshop, participants will be able to:

1. Trace packet flow through the FTD dataplane (ingress to egress).
2. Configure and verify NAT (Static, Dynamic, PAT, Policy, Identity).
3. Build Access Control Policies for Layer 3–7 inspection.
4. Establish and troubleshoot Site-to-Site and Remote Access VPNs.
5. Analyze traffic and resolve issues using packet-tracer, show asp drop, and FMC event logs.
6. Apply real-world troubleshooting methods for NAT, ACP, and VPN scenarios.

## Format

- **Duration:** multiple workshops (customizable lab pacing)
- **Delivery:** Instructor-led with guided hands-on labs
- **Tools Used:** Cisco FMC GUI, FTD CLI, Wireshark, and packet captures
- **Assessment:** Lab exercises, review questions, and troubleshooting challenges
